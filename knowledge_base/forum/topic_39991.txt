It can be set in a python script, see

Python documentation

resource — Resource usage information

This module provides basic mechanisms for measuring and controlling system resources utilized by a program. Availability: Unix, not Emscripten, not WASI. Symbolic constants are used to specify part...

Setting the stack limit to unlimited in your .bashrc is not recommended as it will lead to failures with multi-threaded programs.  If you’re setting this in your .bashrc, double the current limit until things work, but do not set it globally to unlimited.
