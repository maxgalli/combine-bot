REPLY 11 (by kcormier @ 2023-03-20T09:39:35.365Z):
Hi Xingchen,
Yes, combine is really designed for fitting Poisson distributed data, so the likelihood model gets built on this assumption. Unfortunately, there is no easy way to discard that assumption. Note that it can still be used for many tests starting from a MC model despite this (e.g. fits to pseudodata generated from the nominal model). Though it depends on exactly what you’re trying to do of course.
Cheers,
Kyle
