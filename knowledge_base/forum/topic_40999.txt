REPLY 2 (by anigamov @ 2024-05-13T10:32:10.198Z):
Hi Ram,
To submit the Combine jobs to HTCondor you need the combineTool.py which works only with CMSSW.
Once you setup the Combine as described in [1] and compile it with slc7 environment (by running the cmssw-el7 beforehand) the condor submission files should pick up the environment settings correctly.
We are lucky to have some slc7 machines to run the condor jobs, so it should work fine. You can also to set the OS settings manually, as described here.
It is true that you probably can’t run the condor commands from the container, so you have to exit it before running condor_submit <submission_file_name>.sub.
We are working on a combine release compatible with the el9 architecture, so you won’t have to use the slc7 containers in the future.
Best,
Aliya
