REPLY 2 (by kcormier @ 2024-03-01T11:05:50.068Z):
Hi Ravindra,
Although combine is a very useful tool, this is actually one use case it is not really set up for. The tool is really designed for cases where you have a Poisson-distributed count of observations that is the data you are fitting.  But the TProfile point you are using are not poisson distributed observations. The underlying histogram that the TProfile comes from presumably has poisson distributed counts. But I’m not sure that going back to that helps you with your goal. Perhaps someone else can comment if they have a clever idea.
That being said, we are happy to try to help with the tools for this, whether or not you use combine. Do you already have some basic fitting set up?
Best,
Kyle
