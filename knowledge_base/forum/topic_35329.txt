REPLY 4 (by nckw @ 2024-02-22T14:04:24.392Z):
Hi Sasha,
You only need the second command if you need to split the jobs into multiple steps to build the distribution of test statistics.
However, you already have a reasonably accurate result from the first command it seems, which I’ve copied below.
– Hybrid New –
Significance: 1.91416 -0.0351975/+0.0377422
Null p-value: 0.0278 +/- 0.00232496
Done in 34.28 min (cpu), 35.38 min (real)

so you can just take this to be your result from the computation using toys (i.e ~1.9 sigma)
Best,
Nick
