REPLY 2 [ACCEPTED ANSWER] (by gortona @ 2024-11-26T10:59:03.175Z):
Dear Shriniketan,
While it is correct to say the expected limit should improve when adding more data, the same is not necessarily true with the observed limit. If in one year you have a data excess/deficit, this will reflect on the combined limit as well.
You should compare the combined limit with the limits from the 3 years separately, to see if the combined result make sense.
