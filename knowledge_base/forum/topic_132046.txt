QUESTION (by eipearso @ 2025-09-17T18:49:12.294Z):
Dear combine experts,
We are using FitDiangostics to obtain the pre and post b-only fit yields for our analysis, which considers the full Run II dataset. We would like to have the full Run II yield and uncertainty for the signal, each background process separately, and the total background (pre and post-fit). However, the output of FitDiagnostics contains results for each year separately.
Given that some of our systematics are correlated across years, simply adding the uncertainties of each year in quadrature is incorrect, so what is the correct way to combine the uncertainties when we add the years together? We do notice that there is a 2D overall_total_covar matrix included in the output in each folder shapes_prefit and shapes_fit_b, but we are unsure how to use it correctly.
We are running FitDiagnostics with the options shown in the following command, and the datacard and output fitDiagnostics.root file are attached.
combine -M FitDiagnostics datacard.txt -m 300 --robustFit 1 -v1 --rMin -1 --skipSBFit --rMax 2 --saveShapes --saveNormalizations --saveWithUncertainties --saveOverallShapes --setParameters signalScaleParam=0.001316 --trackParameters signalScaleParam --freezeParameters signalScaleParam

One final note: we are part-way through unblinding and have been instructed to not make the s+b fits yet.
Thank you,
Em Pearson
