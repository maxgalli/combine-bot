QUESTION (by mfackeld @ 2022-10-13T13:11:38.443Z):
Dear combine experts,
we noticed very long runtimes when running Fitdiagnostics to extract the uncertainty band for prefit and postfit plots (see attached slides). The runtimes almost scale linearly with the number of toys. Attached you can find the comparison of runtimes and uncertainty bands for 200 toys (default) and 10 toys in the scope of the dileptonic HH → bbWW analysis.
We have the following questions for you:

Is there a recommendation for the number of toys one should use to estimate the prefit/postfit uncertainty bands?
Is it alright to reduce the number of toys from 200 to e.g. 50 for a publication?
Do you know how the time for throwing toys scales with the number of nuisances?

Thank you very much,
Peter for the bbWW analysis team

REPLY 2 (by amarini @ 2022-10-13T13:27:06.699Z):
Hi Manfred,
as usual, when using toys you are introducing an artificial uncertainty on the result due to the statistical fluctuations in toys. 50 is a bit aggressive, IMHO.
The uncertainties on the bands are related to the binomial distribution, and you should have fluctuations with relative size ~ 1./sqrt(N * p * (1.-p) ), which could be correlated among adjacent bins.
For your last question, the time of throwing the toys is pretty negligible.
For throwing post-fit toys, there is a non-negligible time in performing the overall fit, but this could be done only once in advance and the result being reused. The time of fitting the toys is pretty linear in the number of toys. The time of fitting is super linear in the number of systematic uncertainties.
Cheers,
-Andrea

REPLY 3 (by nckw @ 2022-10-13T13:25:20.000Z):
Hi Manfred,
The number of nuisance parameters shouldn’t be a huge issue, the time can scale poorly with number of bins as the uncertainties are calculated by adding to TH1 bin contents the running sum of the variance. There is also the covariance matrix stored (see https://github.com/cms-analysis/HiggsAnalysis-CombinedLimit/blob/102x/src/FitDiagnostics.cc#L884-L887) which you could try to comment out (since I assume you don’t need them) and see if that helps at all?
Generally 100 toys would be ok to calculate a reasonable estimate of the variance for the uncertainty bands but I wouldn’t go too much below that.
Best,
Nick
