REPLY 2 (by agilbert @ 2024-02-19T10:14:24.410Z):
Hi Koushik,
Sorry for the slow reply. Just to check, you should be using the (no longer supported) 102x branch for CombineHarvester if you really need to use CMSSW 10_2_X series. It looks like from the above that you are using the v2.0.0 tag, which is not compatible.
That said, we strongly recommend to use CMSSW 11_3_X and the up-to-date release.
