REPLY 4 (by agilbert @ 2023-07-23T09:22:34.563Z):
Hi Fabio,
The output in your last message is correct - the first row is the best-fit point, then each pair of subsequent rows gives the intervals for each POI in turn. Note in this case for the other POIs just the best-fit is reported again, but they are actually being profiled in the fit.
The issue is you need to keep the same --redefineSignalPOI command in the last step, with the three POIs in the same order. This is so the script knows in which order it should read the intervals output of the limit TTree.
So you should get one json file that contains all the information for all POIs. Then it’s just at the impact plotting step where you choose the POI you want to plot. Can you try that and see if it works?
cheers,
Andrew
