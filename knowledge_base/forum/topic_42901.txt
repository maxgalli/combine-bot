REPLY 7 (by gortona @ 2024-07-19T12:33:33.986Z):
Dear Jingqing,
I will need to refresh my memory a bit, but at first sight your command seems correct to me. You probably need to add the --fullBToys option to get a significant amount of alternative hypothesis entries.
For plotting, can you try this script:
https://github.com/cms-analysis/HiggsAnalysis-CombinedLimit/blob/main/test/plotting/hypoTestResultTree.cxx and check if it suits your needs? Otherwise I will try to dig around and see if I can find the plotting code we used in Run1, but it might take me a while to find it.
About your last question, no POIs and NP are treated differently. When you run LEP-style in each toy it first generates the value of each NP from the prior PDF, and afterward the POI is fitted to get the best fit position of the given toy.
I hope it helps,
Best regards,
Giacomo
