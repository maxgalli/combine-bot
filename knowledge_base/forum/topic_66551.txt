REPLY 2 [ACCEPTED ANSWER] (by amarini @ 2024-11-21T19:51:36.069Z):
kmax *
quoting from the doc
The value * can be specified for imax, which tells Combine to determine the number of observables from the rest of the datacard. In order to better catch mistakes, it is recommended to explicitly specify the value.

This declaration is followed by a specification of the number of background sources to be considered, jmax, and the number of independent sources of systematic uncertainty, kmax:
