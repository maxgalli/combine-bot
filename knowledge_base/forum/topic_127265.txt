REPLY 2 (by amarini @ 2025-07-15T06:27:23.955Z):
Hi Heyon-Seo,
sometimes there is usefulness in discussing things within the physics groups as there is not only expertise on the tools but also insight in the physics behind.
In general, it is not going to be easy to find a set of functions that will satisfy the bias conditions within the Run2 + Run3 strategy; this is also reflected in the choices that entered the previous iterations of the analysis.
For what concern this fit, you are likely doing an “envelope” fit using a set of functional forms which may not be fully optimize or cover the full ph. space.
It is unclear if you are or are not correlating some parameters.
The -s -1 flag add a truly random seed in step2, which is useful if you split toys on multiple jobs. It should yield the same result (within the toys stat. fluctuations) and it should not matter for a single job as you seem doing.
-Andrea
