QUESTION (by snandan @ 2022-05-02T08:01:11.236Z):
Dear Combine experts,
I am running FitDiagnostic with full Run2 data for my analysis and I see it takes more than three days to finish while for limit it takes 2-3 hrs with same data. For FitDiagnostic, I use following command
combineTool.py -M FitDiagnostics datacard --redefineSignalPOIs r --setParameters r=1.0 --setParameterRanges r=-100,100  --saveShapes --saveWithUncertainties --saveNormalization --saveOverallShapes --saveWorkspace --saveToys --saveNLL --cminDefaultMinimizerType Minuit2 --cminDefaultMinimizerStrategy 0 --cminFallbackAlgo Minuit2,1:1.0  --robustHesse 1
while for limit use the command
combineTool.py  -M AsymptoticLimits datacard
It would be nice if someone tells why there is so time difference between limit calculation and postfit plots using same data.
Thanks and Regards,
Saswati
----------------------------------------

REPLY 2 (by amarini @ 2022-05-02T08:27:44.000Z):
The things that are slow are the followings:
–saveWithUncertainties runs several toys to derive the uncertainty bands.
Hesse derives the hessian matrix.
Furthermore, saveWorkspace and saveToys will also take some time (but likely not as much)
Cheers,
-Andrea
