REPLY 2 (by kcormier @ 2024-06-25T19:51:17.257Z):
Hi Sasha and Olga,
I think that in order for the --toysFile argument that you use in your second command to do what you want, you need to also set the --readHybridResults flag as well to tell the algorithm that you’ve already run the hypothesis tests and would like to read the hypothesis test results from that file.
I think this is probably why your second command took so long to run.
Someone should correct me if I’m mistaken, but I think this is the behaviour you’re seeing.
I didn’t check your code, so I don’t know if its doing the right thing or not.
The AltDistribution should be the expected distribution for the signal model pdf, whereas the NullDistribution will be the one for the background model pdf.  In your case, with your specific choice of parameter point, I think these should be the same models for your particular model and choice of point.
Best,
Kyle
